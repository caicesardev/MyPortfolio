1. Moonlight | Ecommerce

Moonlight es un ecommerce en el que estoy trabajando. Su fecha de lanzamiento aún no está prevista ya que actualmente estudio un Ciclo Formativo de Informática y no puedo dedicar mucho tiempo al proyecto.

El proyecto se compondrá de los siguientes elementos:

    1. Tienda 2. Carrito de la compra 3. Página de pagos 4. Página de configuración 5. Perfil del usuario 6. Página de inicio de sesión 7. Página de registro

Tienda

La tienda es donde se mostrarán todos los productos registrados en la base de datos. Cada producto tiene un precio y se puede configurar si es digital o físico.

La tienda está sujeta a cambios y en un futuro se podrá filtrar productos basándose en la categoría e incluso se avisará al usuario si algún producto está en oferta.

Carrito de la compra

En el carrito de la compra se mostrarán todos los productos que el usuario haya añadido.

El usuario podrá añadir y quitar productos libremente y la página le mostrará el precio total de su compra así como el número total de productos seleccionados.

Página de pagos

La página de paogs es donde el usuario verá sus productos y podrá proceder a pagar.

El usuario tendrá que rellenar la información de envío y al continuar se le mostrará las opciones de pago. En un futuro se añadirán nuevas opciones de pago.

Configuración

En la página de configuración el usuario podrá configurar la información de su perfil y parámetros como por ejemplo el tema de colores, las notificaciones y opciones de privacidad.

Aún estoy desarrollando esta página y estoy creando un tema oscuro para la página que se podrá configurar desde aquí. El tema oscuro solo estará disponible para usuarios registrados.

Perfil del Usuario

Esta página mostrará toda la información del usuario así como un historial de pedidos en forma de informe.

Todavía está en desarrollo.

Inicio de Sesión

La página de inicio de sesión permitirá a los usuarios registrados en la página web iniciar sesión.

En un futuro se añadirá una opción para poder recuperar contraseñas en cas de que el usuario se haya olvidado.

Registro

La página de registro permitirá a nuevos usuarios poder registrarse a la página web.

Registrarse automáticamente te permitirá acceder al perfil de usuario y a la página de configuración. Además de futuros beneficios como el autocompletado en la página de pagamiento.

1. PyNotes | PySide6

PyNotes es un editor de texto el cual todavía estoy desarrollando. Este proyecto utiliza PySide6 ya que ya tengo experiéncia con PyQt5 y quería un nuevo desafío. PySide6 ofrece una licencia más permisiva por lo tanto se pueden hacer proyectos comerciales sin comprar una licencia.

Proyecto

PyNotes es un simple editor de texto desarrollado con PySide6 con fines de aprendizaje.

PyNotes contará con las funcionalidades básicas de un editor de texto pero además contará con funcionalidades extras como crear tablas, imprimir el texto, soporte para Markdown, ajustes de texto y mucho más.
